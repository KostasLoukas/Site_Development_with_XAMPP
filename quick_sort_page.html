    Quicksort

    Για πολυπλοκότητα στον χρόνο, έχουμε τρεις περιπτώσεις που μπορούν να μας τύχουν που αντικατοπτρίζουν την πολυπλοκότητα:
    - Στην καλύτερη περίπτωση θα έχουμε: O(nlogn)
    - Σε μία μέση περίπτωση θα έχουμε: O(nlogn)
    - Στην χειρότερη περίπτωση θα έχουμε: O(2n)

    Περιγραφή του αλγόριθμου

    Η QuickSort είναι ένας αλγόριθμος Διαιίει και Βασίλευε. Διαλέγει ένα στοιχείο ως περιστρεφόμενο και χωρίζει τον δεδομένο πίνακα γύρω από τον περιστρεφόμενο άξονα. Υπάρχουν πολλές διαφορετικές εκδόσεις του QuickSort που επιλέγουν το άξονα με διαφορετικούς τρόπους.

    Ψευδοκώδικας:

     
                  # Eπιλέξτε περιστροφή
              swap a[1,rand(1,n)]
              
              # Aμφίδρομo χώρισμα
              k = 1
              for i = 2:n, if a[i] < a[1], swap a[++k,i]
              swap a[1,k]
              → αμετάβλητο: a[1..k-1] < a[k] <= a[k+1..n]_
              
              # αναδρομικά ταξινόμηση
              sort a[1..k-1]
              sort a[k+1,n]
